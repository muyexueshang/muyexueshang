DISCUZ_VERSION：Discuz_X3.4_SC_UTF8
在windows下解压DISCUZ压缩包，找到upload
将upload移动到/root
CENTOS7 IP 10.1.15.151
下面以root身份操作
yum install -y httpd.x86_64
yum install -y mariadb-server.x86_64
yum install -y php-mysql #mysqli_connect()不支持advice_mysqli_connect错误
cd /root/
cp upload/ /var/www/html/ –rf
cd /var/www/html/
chmod 777 upload –Rf
systemctl start mariadb
systemctl start httpd
在windows网页中访问 本机IP/upload
点击“我同意”
点击“下一步”，全新安装，“下一步”
修改 数据库用户名（给个自定义用户），密码为空
修改管理员帐号、密码、Email、系统信箱 点击“下一步”
